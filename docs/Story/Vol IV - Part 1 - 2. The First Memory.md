# 2. The First Memory
**Volume**: [[Volume IV – The Core and the Final Choice]] | **Part**: [[Part 1 – The Truth of the Core]]

---

As [[Christ]] steps forward, the first memory surrounds him.
He no longer watches history.
He lives it.
He sees humanity before the Seven existed.
Cities filled with brilliant inventors.
Artists.
Scientists.
Philosophers.
People capable of incredible kindness...
and incredible cruelty.
Humanity flourished.

Then ambition became greed.
Kings sought immortality.
Empires sought absolute control.
Scholars attempted to command Time itself.
Others tried to erase Death.
Some wished to create endless life.
Others wanted limitless destruction.
The Fundamental Laws were treated as tools.
Eventually...
humanity discovered [[The Core|the Core]].
